# 网卡开机
## 第一步 安装工具
- 安装ethtool工具，并用此工具在软件层面开启网卡唤醒功能
```bash
sudo apt update
sudo apt install ethtool
```
---
## 第二步 使用工具
1.**检查wol状态**
```bash
sudo ethtool [你的网卡名] | grep -E "Supports Wake-on|Wake-on"
```
> Supports Wake-on: pumbg ：表示你的硬件支持这些唤醒方式\
> Wake-on: d 或 Wake-on: g ：这是当前的激活状态。d 表示禁用（Disabled），g 表示已启用，监听“魔法包”（Magic Packet）模式

> ✅ 如果输出 Wake-on: g，说明网卡 WOL 已启用。\
> ❌ 如果输出 Wake-on: d 或其他字母，则需要手动启用。
---

2. **手动开启网卡唤醒（仅一次）**
```bash
sudo ethtool -s [你的网卡名] wol g
```
---
3.  永久化(采用systemctl，每次开机执行一次命令）
> 步骤：\
> 创建服务文件\
> 设置开机自启

```bash
sudo vim /etc/systemd/system/wol-enable.service

[Unit]
Description=Enable Wake-on-LAN

[Service]
Type=oneshot
ExecStart=/usr/sbin/ethtool -s enp2s0 wol g

[Install]
WantedBy=multi-user.target

sudo systemctl enable wol-enable.service

```
### 至此软件层面配置完成，接下来要在bios中开启网卡开机功能

## 第三步 bios开启网卡唤醒
- 按F2或者Del进入biso（根据主板厂商决定
- 一般在‘高级’选项中 开启以下选项
- 板载设备配置 (OnBoard Device Configuration)[Enabled]网卡硬件总开关
- 板载设备配置	网卡8102EROM控制	[Enabled]允许网卡在关机后待命
- 电源管理/APM设置 (Power Management / APM)Power Management/APM	[Enabled]高级电源管理总开关
- 电源管理/APM设置Wake-Up by PME[Enabled]	最关键：允许 PCIe 设备（网卡）唤醒主板
