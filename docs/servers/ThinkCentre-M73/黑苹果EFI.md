# ThinkCentre M73 Hackintosh EFI
> 20260620 保存黑苹果efi,打算换掉硬盘,装arch玩,故此保存EFI信息

![M73前面板图](https://user-images.githubusercontent.com/849044/82004638-46d58000-9618-11ea-830b-aca33298657d.jpg)

## 硬件配置
- CPU : i3-4170   
- GPU : Intel HD Graphics 4400 1536 MB  
- 内存: 2*4G 1600 MHz DDR3  
- 网卡: AX210  

## 使用情况
- Catalina 10.15.7 ✅  (其余系统未尝试)  
- 隔空投送 ✅
- 取消勾选‘如果可能,使硬盘进入睡眠’避免睡眠无法唤醒 🫤🫤    ( 🍎 -->系统偏好设置-->节能)  
- 开机某一瞬间音响口有杂音暴响 😡
